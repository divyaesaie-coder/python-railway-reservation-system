# python-railway-reservation-system
A simple Python railway reservation system to book tickets, cancel tickets, manage waiting lists, and track available seats.
# Python Railway Reservation System

A beginner-friendly Python project that simulates a railway ticket reservation system.

## Features
- Book train tickets
- Cancel booked tickets
- View remaining seats
- Manage waiting list
- Menu-driven console application

## Technologies Used
- Python 3

## How It Works
The program starts with a fixed number of seats and allows users to:
1. Book tickets
2. Cancel tickets
3. View available seats
4. View waiting list
5. Exit the system

If seats are unavailable, users are added to the waiting list automatically.

## Concepts Used
- while loop
- if-else conditions
- Variables and counters
- User input handling
